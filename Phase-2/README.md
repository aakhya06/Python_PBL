Phase 2 – Model Training and Evaluation
Project Title:
Diabetes Risk Prediction using Machine Learning
Objective:
The objective of this phase is to implement machine learning techniques on the preprocessed dataset, train predictive models, and evaluate their performance for diabetes risk classification.
Description:
In this phase, the project focuses on applying machine learning algorithms to the preprocessed dataset to build predictive models. The dataset is first prepared by selecting the most relevant features using correlation analysis. It is then split into training and testing sets to ensure proper evaluation. Missing values are handled using median imputation, and feature scaling is applied using StandardScaler to normalize the data.
Multiple machine learning algorithms are implemented, including Random Forest Classifier, Logistic Regression, and K-Nearest Neighbors (KNN). These models are trained on the selected features to learn patterns from the dataset and predict diabetes risk. Visualization techniques such as boxplots and correlation heatmaps are used to analyze feature relationships and data distribution. This phase ensures that the models are trained on clean, structured, and meaningful data.
Tasks Completed:
Feature selection using correlation analysis
Splitting dataset into training and testing sets
Handling missing values using SimpleImputer
Feature scaling using StandardScaler
Model training using Random Forest Classifier
Model training using Logistic Regression
Model training using K-Nearest Neighbors (KNN)
Data visualization (boxplots and heatmap)
Initial model implementation and testing
Tools & Technologies Used:
Python, Pandas, NumPy, Matplotlib / Seaborn, Scikit-learn
Files Included:
db_risk.csv – Preprocessed dataset
PBL_notebook.ipynb – Model training and implementation code
Output Screenshots – Visualizations and model outputs
Outcome of Phase-2:
The machine learning models are successfully trained using the preprocessed dataset, and important features influencing diabetes prediction are identified. Multiple algorithms are implemented to learn patterns from the data, making the system ready for performance evaluation, model comparison, and optimization in the next phase.
