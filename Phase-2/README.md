Phase 2 – Model Training and Evaluation
Project Title:
Diabetes Risk Prediction using Machine Learning

Objective:
The objective of this phase is to implement machine learning techniques on the preprocessed dataset, train a predictive model, and evaluate its performance for diabetes risk classification.

Description:
In this phase, the project focuses on applying machine learning algorithms to the preprocessed dataset to build a predictive model. The dataset is first prepared by selecting the most relevant features using correlation analysis. It is then split into training and testing sets to ensure proper evaluation. Feature scaling is applied using StandardScaler to normalize the data.
A Random Forest Classifier is implemented to train the model on the selected features. The model learns patterns from the training data and is used to predict diabetes risk. Visualization techniques such as boxplots and correlation heatmaps are used to analyze feature relationships and data distribution. This phase ensures that the model is trained on clean, structured, and meaningful data.

Tasks Completed:
Feature selection using correlation analysis
Splitting dataset into training and testing sets
Feature scaling using StandardScaler
Model training using Random Forest Classifier
Data visualization (boxplots and heatmap)
Initial model implementation and testing

Tools & Technologies Used: Python, Pandas, NumPy, Matplotlib / Seaborn, Scikit-learn

Files Included:
db_risk.csv – Preprocessed dataset
PBL_notebook.ipynb – Model training and implementation code
Output Screenshots – Visualizations and model outputs

Outcome of Phase-2:
The machine learning model is successfully trained using the preprocessed dataset, and important features influencing diabetes prediction are identified. The system is now ready for further evaluation, comparison with other models, and performance optimization in the next phase.
