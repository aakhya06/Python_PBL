Phase 1 – Project Planning, Synopsis and Data Preprocessing
Project Title: Diabetes Risk Prediction using Machine Learning

Objective:
The objective of this phase is to understand the problem of diabetes prediction, prepare the project synopsis, generate a custom dataset, and perform initial data preprocessing required for machine learning implementation.

Description:
In this phase, the project focuses on designing a system for predicting diabetes risk using machine learning. A synthetic dataset of 1000 patient records is generated using Python with medically realistic feature ranges such as Glucose, Blood Pressure, BMI, Insulin, Age, and Pregnancies. A Patient_ID is assigned to each record, and the target variable (Outcome) is created using a risk-based approach. Data preprocessing steps including handling missing values, adding noise, and organizing the dataset are performed to simulate real-world conditions and prepare the data for model training.

Tasks Completed:
Problem identification and project planning
Synopsis preparation
Dataset creation using Python
Data preprocessing (missing values, noise addition)
Initial data verification and analysis
Flowchart design for system workflow
