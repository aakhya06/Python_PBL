# Python_PBL
Diabetes Risk Prediction :
This project focuses on building a machine learning system to predict whether a patient is diabetic or non-diabetic based on clinical health parameters. A custom synthetic dataset is generated using Python with medically realistic ranges for features such as Glucose, Blood Pressure, BMI, Insulin, Age, and Pregnancies.
In Phase 1, the project includes problem analysis, synopsis preparation, dataset generation, and data preprocessing (handling missing values, adding noise, and organizing data). A system flowchart is also designed to represent the complete machine learning pipeline.
The project will further implement classification models such as Logistic Regression, KNN, and SVM to evaluate and compare performance in predicting diabetes risk.

Objective: To develop an accurate and reliable system for early diabetes prediction using machine learning techniques.
